"TP DWWM" : présentation

Le "Titre Professionnel : Développeur Web et Web Mobile" est une formation professionnalisante de type "full stack", articulée en 2 parties : front-end et back-end. Chaque partie est composée de 4 modules. La certification obtenue après examen devant un jury de professionnels est de niveau 5 (Bac+2, BTS, DUT).

#1 : front-end

1. Maquetter une application
2. Réaliser une interface utilisateur web statique et adaptable
3. Développer une interface utilisateur web dynamique
4. Réaliser une interface utilisateur avec une solution de gestion de contenu ou e-commerce

#2 : back-end

5.Créer une base de données
6. Développer les composants d'accès aux données
7. Développer la partie back-end d'une application web ou web mobile
8. Elaborer et mettre en oeuvre des composants dans une application de gestion de contenu ou e-commerce
